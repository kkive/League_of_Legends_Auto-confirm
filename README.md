# League_of_Legends_Auto-confirm
基于yolov8模型完成英雄联盟自动接受进入游戏
需要这个功能已经融合到[无糖管家](https://github.com/kkive/Free-Suger_Butler)，欢迎大家来体验。  
# 功能
1.英雄联盟自动确认
2.进游戏自动暂停识别，重新匹配时开始识别
